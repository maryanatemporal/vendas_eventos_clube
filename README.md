📊 Análise de Vendas de Eventos

Projeto de análise de dados desenvolvido em Python utilizando Pandas para analisar o faturamento de diferentes eventos.
O objetivo do projeto é organizar e analisar dados de vendas, identificando quais eventos geram maior faturamento.

🛠 Tecnologias utilizadas
Python
Pandas
Google Colab

📂 Estrutura do projeto
vendas_eventos
│
├── vendas_eventos.csv
├── gerentes.xlsx
├── analise_vendas_eventos.ipynb
└── README.md

📈 Análises realizadas:

Importação de dados em CSV e Excel

Tratamento e organização dos dados

Agrupamento de faturamento por evento utilizando groupby()

Cálculo da soma do faturamento com sum()

Ordenação dos resultados do maior para o menor faturamento

🎯 Objetivo do projeto:
Praticar conceitos de análise de dados com Python e Pandas, aplicando manipulação, tratamento dos dados e análise.
